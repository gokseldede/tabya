1ki3 Canlı Destek

    Kurulum yapılandırma

    Burada bazı temel bilgilerinizi yapılandırabilirsiniz. Bu sayfada tek yönetici parolası ve agent parolası belirlenecektir. Agent istediği isim ile giriş yapabilir. Şifre agentler için ortak olarak atanmaktadır. Eğer yönetici şifresini unutursanız App_Data klasöründeki "config" dosyasını silmeniz yeterlidir.

    Şifrelerinizi ayarlayın

    Tüm yöneticiler aynı parolayı paylaşır. Yöneticiler için şifre ortaktır.

    Tüm agentlar aynı parolayı paylaşır. Agentlar için parola ortaktır.
